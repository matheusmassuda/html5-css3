Aprendendo CSS

CSS básico - identificadores

A grande vantagem de se usar CSS3 é seu efeito em cascata, onde podemos estilizar várias elementos “iguais” de uma só vez. Por exemplo, quando escrevemos uma tag e passamos o parâmetro de background-color para ela, todas as tags iguais a ela também serão alteradas. Podemos resolver isso passando um parâmetro id = identificator para essa tag. Identificadores são representados pelo # (hashtags / jogo da velha) e só existem um com cada valor.

CSS intermediário

A representação em CSS3 de um id (identificator) em CSS3 fica: 

            
        h1#primeiro-h1 {
            text-align: center;
        }

Podemos também colocar mais do que um valor no parâmetro class, o que significa múltiplos estilos no mesmo objeto apenas separando os valores com um espaço, o código ficaria parecido com: 

            
                <span class="basico marca-texto">
