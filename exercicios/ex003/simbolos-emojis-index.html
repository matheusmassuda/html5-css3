Símbolos e Emojis

Não é possível colocar o sinal de menor < e o sinal de maior > dentro de um parágrafo da forma converncional Shift + , e Shift + .

Pois isso implicaria em criar uma <Tag Fictícia> que o navegador tentaria ler como uma <Tag Comum>.

Criando os sinais de Maior > e Menor <

Para criar o sinal de maior basta escrever &+g+t+; que vem de GT que do inglês significa Greater than.

Para criar o sinal de menor basta escrever &+l+t+; que vem de LT que do inglês significa less than.

Outros Símbolos

&+r+e+g+; para fazer ®: o símbolo de marca registrada.

&+c+o+p+y; para fazer ©: o símbolo de Copyright.

&+t+r+a+d+e; para fazer ™: o símbolo de marca registrada em inglês.

&+e+u+r+o;e&+p+o+u+n+d+;e&+y+e+n+;e&+c+e+n+t+; para fazer as moedas € , £ , ¥ , ¢: respectivamente os símbolos de moeda para Euro, Libras, Yen e centavos de dollar.

&+D+e+l+t+a+; para fazer Δ: o símbolo de Delta.

&+u+p+a+r+r+o+w; para fazer ↑: o símbolo de Seta para cima.

&+l+e+f+t+a+r+r+o+w; para fazer ←: o símbolo de Seta para cima.

Adicionando Emojis

Para adicionar emojis é interessante usar o emojipedia

🖖🤓 Escrevendo o código do emoji diretamente de emojipedia temos mais garantia dele funcionar em qualquer navegador.

O emoji anterior da mão "🖖" possui o seguinte código:&+#+x+1+F+5+9+6+; o #+x indica que ele é um código hexadecimal, assim como uma cor, por exemplo.
