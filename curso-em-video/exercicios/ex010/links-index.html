Usando links externos

Usando uma <a> no html você pode criar um link que leva para o Canal do Youtube do Skilu

Parâmetros que são muito importantes na <a>

  • href (Hypertext reference): endereço de URL
  • target (alvo): abre página numa nova guia "_blank"
  • rel (relative): "external"

Esses parâmeros são muito importantes para SEO.
