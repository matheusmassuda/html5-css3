O Google Fonts é um repositório externo, confiável e gratuito de fontes que podemos importar para nosso arquivo CSS3. Podemos usar links <link> para nosso documento HTML5 ou importações @import para nossos arquivos de CSS3. Para incorporá-las basta acessar Google Fonts, escolher a fonte desejada e clicar na opção (embed code - incorporar código) Depois de importadas as fontes começam a funcionar normalmente, inclusive na parte de embed code do Google Fonts ele também sujere uma maneira de escrever o font-family
