Título de External Font

Podemos baixar uma fonte de um site como DaFont e extrair os arquivos com extensões .otf ou .ttf dentro de uma pasta chamada (fonts) irmã de index.html. Desse modo temos o arquivo do tipo fonte que permite "recriar" essa fonte dentro do arquivo HTML5

Para recriar essa fonte dentro do nosso documento HTML5, primeiro precisamos criá-la em nosso arquivo de estilização CSS3 (style.css). O código fica parecido com:

        
            @font-face {
                font-family: love cursive;
                src: url(fonts/lovemy.otf) format('opentype'), url(fonts/lovemy.ttf) format('truetype');
                font-weight: normal;
                font-style: normal;
            }

@font-face é o comando > font-family é o nome que queremos "cadastrar" aquela fonte > url ( ) é o caminho onde aquele arquivo de fonte está > format ( ) é o formato da font que precisa ser colocado.
