Entendendo Flexbox

Para todos os exemplos, usaremos a seguinte estrutura:

    <div class="container">
        <div class="item"> A </div>
        <div class="item"> B </div>
        <div class="item"> C </div>
        <div class="item"> D </div>
    <div class="container">

flex-direction: row (linha)

main-axis: eixo x da esquerda para direita // cross-axis eixo "y"de cima para baixo

A
B
C
D

flex-direction: row-reverse (linha reversa)

main-axis: eixo x da direita para esquerda // cross-axis eixo "y"de cima para baixo

A
B
C
D



flex-direction: column (coluna)

main-axis: eixo y de cima para baixo // cross-axis eixo "x" da esquerda para direita

A
B
C
D

flex-direction: column-reverse (coluna reversa)

main-axis: eixo y de baixo para cima // cross-axis eixo "x" da esquerda para direita

A
B
C
D
