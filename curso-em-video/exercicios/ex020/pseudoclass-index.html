Pseudo Classes

Pseudo classes são usadas para relacionar uma ação para um determinado estado de um elemento. Os estados mais comuns são: hover (mouse em cima), ativo, marcado, desabilitado. Quando um determinado elemento estiver "sofrendo" aquele determinado estado a pseudo classe irá devolver a ação.

O código em CSS3 fica parecido com: 

            
        
            div:hove{
                background-color:blue;
            }
E deve ser lido: Para as divs ":"> no estado hover > aplique background azul.
